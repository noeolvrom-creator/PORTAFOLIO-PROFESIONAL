# Automated Mixing Machine – Control System

## Overview
Control system developed for an automated mixing machine,
focused on process control and industrial communication.

## Hardware
- Allen Bradley PLC
- Arduino board
- Flow and temperature sensors
- Actuators and heating elements

## Software
- PLC programming (Allen Bradley)
- HMI development using FactoryTalk View
- Arduino firmware for sensor acquisition

## Communication
- Modbus communication between PLC and Arduino
- Serial communication for sensor data acquisition

## Key Features
- PID control for temperature and flow regulation
- Human–Machine Interface for process monitoring
- Integration of industrial and embedded devices

## Testing & Validation
- Functional testing of control loops
- Parameter tuning for stable process operation

## Notes
This project focused on industrial control, communication protocols,
and system validation.
